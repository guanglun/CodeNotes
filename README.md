## CodeNotes(代码笔记)   

* 添加注释
* 代码标记
* 快速跳转

## 快速使用

* 打开一个工作区，在该工作区初始化CodeNotes(会在该工作区创建.codenotes文件夹存储标记)
* 在需要做标记的位置(选择需要做标记的文件内容)右击CodeNotes Insert Mark将插入一则标记
* 右侧可编辑标记名称、颜色、描述。描述内容支持markdown，将在鼠标在选择的文字上放置时显示
